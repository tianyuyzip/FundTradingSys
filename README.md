# FundTradingSys
本系统满足了开发式基金的销售问题，基金销售人员可利用WEB使用它来和他们的客户进行基金交易，并对基金产品、客户资料和客户资金帐户进行了管理。

### 本系统是一个关于开发式基金进行交易的平台，它主要包括一下功能 ：
1. 基金产品管理
2. 客户资料管理
3. 客户资金账户管理
4. 基金交易管理
  
### 用户环境
+ 该银行的数据库使用的是Oracle9i。
+ 应用服务器为Weblogic8.1

### 系统性能
+ 至少可100人并发访问，WEB页面相应不超过3秒。

###	技术要求
+ 系统采用J2EE技术，利用Struts、EJB和WebService来实现此平台。使系统基于组件，可扩展性强，并能分布式部署
